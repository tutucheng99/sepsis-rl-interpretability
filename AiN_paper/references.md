## 格式说明 (AiN Vancouver Style)

### Journal Article:
```
Author1 AB, Author2 CD, Author3 EF, et al. Title of article. J Abbrev. Year;Volume(Issue):Pages. doi: xxx.
```

### Book:
```
Author AB. Book Title, Edition. Publisher: City, Country; Year. pp. Pages.
```

### Conference:
```
Author AB, Author CD. Title of presentation. In: Proceedings of Conference Name; Year. pp. Pages.
```

### Website:
```
Title of Site. Available online: URL (accessed on Day Month Year).
```
