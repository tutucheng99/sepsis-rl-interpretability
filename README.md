## Abstract

**Background/Objectives.** Sepsis is a leading cause of mortality in critical care and is associated with substantial long-term neurological morbidity among survivors. Reinforcement learning (RL) offers a route to data-driven treatment policies, but clinical adoption is limited by concerns that interpretability must be sacrificed for performance and that interactive (online) training is inherently superior to learning from static (offline) data. We benchmark these assumptions by comparing outcome performance and quantitative interpretability across offline and online RL approaches for sepsis treatment.

**Methods.** Using the gym-sepsis simulator derived from MIMIC-III intensive care electronic health records, we trained offline policies with Behavior Cloning (BC) and Conservative Q-Learning (CQL) on a fixed dataset of 2,105 simulated episodes (approximately 20,000 transitions) generated by a guideline-inspired heuristic policy. We also trained three online methods (DDQN-Attention, DDQN-Residual, SAC) with simulator interaction, and included a vanilla DQN baseline trained via interaction. Policies were evaluated on 500 newly sampled episodes per method, reporting overall survival and survival stratified by initial SOFA severity. Interpretability was quantified via Linearly Estimated Gradients (LEG) feature-saliency analysis.

**Results.** Survival rates were tightly clustered (94.0%–95.4%) across learned policies. For high-severity episodes (SOFA ≥ 11), DDQN-Attention reached 90.5% survival vs. 88.5%–88.6% for CQL and BC; vanilla DQN underperformed at 84.3%. Interpretability differed substantially: CQL produced the strongest saliency (max 40.06, emphasizing systolic blood pressure), online methods were intermediate (1.17–3.57), BC was lower (0.78), and DQN was negligible (0.069).

**Conclusions.** Online methods provided at most marginal survival improvements, while conservative offline learning achieved comparable outcomes with markedly stronger and more clinically coherent feature-importance signals and without requiring exploratory interaction during training.

**Keywords:** Reinforcement learning · Sepsis treatment · Interpretability · Conservative Q-learning · LEG analysis · Offline RL · MIMIC-III · Sepsis-associated encephalopathy · Neurological outcomes · Blood-brain barrier · AI in neurology

```

  create_6model_leg_comparison.py   Generate Figure 3 (LEG 6-model comparison)
```

## Reproduction

```bash
# Environment
conda create -n sepsis_rl python=3.10
conda activate sepsis_rl
pip install -r requirements.txt
pip install -e gym-sepsis

# Evaluate all methods (500 episodes each)
python scripts/re_evaluate_all.py

# LEG interpretability analysis
python scripts/Interpret_LEG/leg_analysis_offline.py
python scripts/Interpret_LEG/leg_analysis_online.py

# Regenerate main figures
python scripts/06_visualization.py
python scripts/create_6model_leg_comparison.py
```

## Data Availability

All data derive from publicly available sources. The gym-sepsis simulator is based on the MIMIC-III database (https://physionet.org/content/mimiciii/); access requires completion of PhysioNet data-use agreements and training. No new patient-level data were generated in this study.
